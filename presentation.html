2.4 — I/O Pad (Open-Drain)

scl_o    = 1'b0                 // luôn drive LOW khi enabled
scl_oe_o = ~scl_out_q & i2c_en // OE=1 → kéo LOW; OE=0 → thả (pull-up)

2.5 — FSM điều khiển I2C (14 states)

ST_IDLE → ST_START_A → ST_START_B
       → ST_WRITE_BIT ↔ ST_WRITE_WAIT → ST_ACK_READ → ST_ACK_RWAIT
       → ST_READ_BIT  ↔ ST_READ_WAIT  → ST_ACK_SEND → ST_ACK_WAIT
       → ST_STOP_A    → ST_STOP_B     → ST_STOP_C   → ST_IDLE

🛡️ CHUẨN BỊ TRẢ LỜI CÂU HỎI KỸ THUẬT

❓ Q1: OBI Bus Protocol là gì? Tại sao không dùng AXI hay APB?

OBI — Open Bus Interface — do PULP Platform phát triển, tối ưu cho SoC nhỏ RISC-V. Chỉ cần req, gnt, rvalid — đơn giản hơn AXI nhiều. SoC CROC đã dùng OBI sẵn, nên giữ nhất quán. Dùng APB phải thêm bridge → tốn area.

❓ Q2: I2C FSM hoạt động như thế nào?

14 states, mỗi SCL cycle chia 5 phase. Flow WRITE: IDLE → START_A/B → WRITE_BIT/WAIT (×8) → ACK_READ/RWAIT → IDLE hoặc STOP. Prescaler: PRE = Fclk / (5 × Fscl) − 1. Ví dụ: 20MHz / (5 × 100kHz) − 1 = 39.

❓ Q3: Timing closure — tại sao WNS = 0?

WNS = 0 = không path nào vi phạm. Worst slack +1.23 ns. Critical path qua debug module, không phải I2C. CTS + post-route optimization tự chèn buffer. Final: 0 setup/hold violations.

❓ Q4: Tại sao chọn IHP 130nm thay vì SkyWater 130nm?

(1) CROC được thiết kế sẵn cho IHP — có scripts, constraints, pad ring. (2) IHP SG13G2 có standard cell library hoàn chỉnh với multi-corner timing. (3) IHP hỗ trợ MPW miễn phí cho nghiên cứu/giáo dục.

❓ Q5: Lỗi slew/fanout/cap violations?

Chủ yếu ở I/O pad connections (capacitance ~15pF). Không ảnh hưởng timing closure (WNS/TNS = 0). Production sẽ thêm I/O buffer cells.

❓ Q6: Arbitration Lost trong sim?

2-stage synchronizer gây trễ 2 clock → master đọc SDA cũ → false positive. Trên HW thật, I2C clock (100kHz) rất chậm so với sys clock → không xảy ra. Dùng sim_wait_tip() workaround.

❓ Q7: 3 DRC violations fix được không?

Metal Spacing trên Metal4, gần SRAM macro. Fix bằng tăng halo hoặc manual ECO. Trong academic context, 3 minor violations trên chip 3.67 mm² là chấp nhận được.
